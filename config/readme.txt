Configuration Information

No modifications in this folder are required.
If you need to update model or training configurations, please refer to and modify the following files accordingly:

Model Configuration: Verse_1_0_Cardiac/configs/Cardiac_config.py
Training Configuration: Verse_1_0_Cardiac/configs/Training_config.py
Testing Configuration: Verse_1_0_Cardiac/configs/Testing_config.py

For any adjustments, please ensure that changes are made in the appropriate configuration files.
